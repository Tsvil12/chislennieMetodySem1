# Метод Гаусса с выбором главного элемента

## 📌 Что это за метод?

Метод Гаусса — это **прямой метод** решения систем линейных уравнений. Он состоит из двух этапов:

1. **Прямой ход** — приводим матрицу к треугольному виду
2. **Обратный ход** — находим неизвестные

**Выбор главного элемента** — это трюк, который помогает избежать деления на ноль и уменьшает ошибки округления.

## 📖 Формула из методички

**Прямой ход (стр. 36):**
\[
d_{kj} = \frac{a_{kj}^{(k-1)}}{a_{kk}^{(k-1)}}, \quad a_{ij}^{(k)} = a_{ij}^{(k-1)} - d_{kj} a_{ik}^{(k-1)}
\]

**Обратный ход (стр. 37):**
\[
x_n = g_n, \quad x_i = g_i - \sum_{j=i+1}^{n} d_{ij} x_j
\]
