# Лабораторная работа №2 (Часть 2): Интерполяция полиномом Гаусса (n=2)

## Задание
Из методички, стр. 15, задание 2:
> "Используя полином Гаусса при n=2, найти значение функции y = f(x) в точке x = x0 с заданной точностью ε."

**Вариант:** №6 из списка функций — `cos(sin x²)`

**Параметры:**
- Функция: `f(x) = cos(sin²x)`
- Отрезок: `[0, π]`
- Точка: `x0 = 1.5`
- Точность: `ε = 1e-10`

---

## Теоретические основы (по методичке)

### Интерполяционная формула Гаусса для n=2 (стр. 11-12)

**Конечные разности (стр. 11):**
\[
y^1_{j+1/2} = y_{j+1} - y_j, \quad y^2_j = y_{j-1} - 2y_j + y_{j+1}
\]

**Формула Гаусса (стр. 12):**
\[
f(x) = y_j + y^1_{j+1/2} \cdot t + y^2_j \cdot \frac{t(t-1)}{2!} + R_2(x), \quad t = \frac{x - x_j}{h}
\]

**Остаточный член (стр. 12):**
\[
R_2(x) = f^{(3)}(\xi) h^3 \frac{t(t^2 - 1)}{3!}, \quad \xi \in [x_{j-1}, x_{j+1}]
\]

```python
# ============================================================================
# Строки 1-3: Импорт библиотек
# ============================================================================
import numpy as np
import matplotlib.pyplot as plt

# ============================================================================
# Строки 5-6: Определение интерполируемой функции
# ============================================================================
def f(x):
    return np.cos(np.sin(x)**2)
# Соответствие: Вариант №6, стр. 15

# ============================================================================
# Строки 8-12: Параметры задачи
# ============================================================================
a = 0                      # Левая граница отрезка
b = np.pi                  # Правая граница отрезка
n = 2                      # Степень полинома (n=2 по условию)
x0 = 1.5                   # Точка, в которой ищем значение
epsilon = 1e-10            # Заданная точность

print(f"Функция: y = cos(sin²x)")
print(f"Отрезок: [{a}, {b}]")
print(f"Точка x0 = {x0}")
print(f"Точность ε = {epsilon}")
print(f"n = {n}")
print()

# ============================================================================
# Строки 15-20: Узлы Гаусса для n=2
# Таблица 3.1, стр. 19: для n=2: t = ±1/√3
# ============================================================================
t1 = -1/np.sqrt(3)         # Первый узел Гаусса на [-1, 1]
t2 = 1/np.sqrt(3)          # Второй узел Гаусса на [-1, 1]

# Преобразование узлов с [-1,1] на [a,b]
# Формула: x_i = (b+a)/2 + (b-a)/2 · t_i  (стр. 19)
x1 = (b - a)/2 * t1 + (a + b)/2
x2 = (b - a)/2 * t2 + (a + b)/2

print("Узлы Гаусса (на [a,b]):")
print(f"x1 = {x1:.6f}")
print(f"x2 = {x2:.6f}")

# ============================================================================
# Строки 22-25: Значения функции в узлах Гаусса
# ============================================================================
y1 = f(x1)
y2 = f(x2)

print("\nЗначения функции в узлах:")
print(f"f(x1) = {y1:.6f}")
print(f"f(x2) = {y2:.6f}")

# ============================================================================
# Строки 27-32: Базисные полиномы Лагранжа
# Формула: L_i(x) = Π (x - x_j)/(x_i - x_j), j≠i
# ============================================================================
def L1(x):
    return (x - x2)/(x1 - x2)    # Базис для первого узла

def L2(x):
    return (x - x1)/(x2 - x1)    # Базис для второго узла

# ============================================================================
# Строки 34-35: Интерполяционный полином Гаусса
# P(x) = y1·L1(x) + y2·L2(x)
# ============================================================================
def G(x):
    return y1 * L1(x) + y2 * L2(x)

# ============================================================================
# Строки 37-42: Вычисление значения в точке x0
# ============================================================================
exact_value = f(x0)               # Точное значение функции
gauss_value = G(x0)               # Приближённое значение по Гауссу
error = abs(exact_value - gauss_value)  # Погрешность

print(f"\nРЕЗУЛЬТАТ в точке x0 = {x0}:")
print(f"Точное значение:  {exact_value:.6f}")
print(f"По Гауссу (n=2):  {gauss_value:.6f}")
print(f"Погрешность:      {error:.6f}")

# ============================================================================
# Строки 45-60: Построение графиков (согласно заданию 2, стр. 15)
# ============================================================================
x_plot = np.linspace(a, b, 100)   # Тонкая сетка для графика
y_exact = f(x_plot)               # Точная функция
y_gauss = [G(x) for x in x_plot]  # Полином Гаусса

plt.figure(figsize=(10, 6))
plt.plot(x_plot, y_exact, 'b-', linewidth=2, label='cos(sin²x) - точная')
plt.plot(x_plot, y_gauss, 'r--', linewidth=2, label='Полином Гаусса (n=2)')
plt.scatter([x1, x2], [y1, y2], color='green', s=80, zorder=5, label='Узлы Гаусса')
plt.scatter([x0], [exact_value], color='black', s=100, zorder=5, label=f'x0 = {x0}')
plt.axvline(x=x0, color='gray', linestyle=':', alpha=0.7)

plt.title('Интерполяция полиномом Гаусса (n=2)')
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
