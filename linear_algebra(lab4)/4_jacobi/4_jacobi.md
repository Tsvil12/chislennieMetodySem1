# Метод Якоби (метод простой итерации)

##  Что это за метод?

Метод Якоби — это **итерационный метод** решения СЛАУ. Вместо того чтобы решать систему точно (как Гаусс), мы **постепенно уточняем** решение.

**Основная идея:** Из каждого уравнения выражаем x_i через остальные и итерационно обновляем.

##  Формула из методички

**Приведение к виду x = Bx + C (стр. 39):**
\[
x_i = \frac{1}{a_{ii}} \left( f_i - \sum_{j \neq i} a_{ij} x_j \right)
\]

**Итерационная формула (стр. 39):**
\[
x_i^{(k+1)} = \sum_{j=1}^{n} b_{ij} x_j^{(k)} + c_i
\]

**Условие сходимости (стр. 39):**
\[
|a_{ii}| > \sum_{j \neq i} |a_{ij}|
\]
